# Prospector de Sites — Claude Skills

Plugin para **Claude** (Cowork / Claude Code) v0.15.0 — ciclo completo de prospecção e venda de sites.

**Achou → Refez (ou criou do zero) → Publicou → Ofertou** (+ respostas, follow-up, contrato)

## Comandos

| Comando | Função |
|---------|--------|
| `/setup` | Configuração inicial + dashboard + publicador |
| `/prospectar` | Busca leads no Google Maps (site fraco ou sem site) |
| `/redesenhar` | Redesign premium (leads com site, mín. 5 leads) |
| `/criar-site` | Site do zero (leads sem site, mín. 5 leads) |
| `/editor` | Editor visual no navegador |
| `/publicar` | Deploy Hostinger + página-capa |
| `/proposta` | E-mail e WhatsApp anti-spam com link da capa |
| `/respostas` | Verifica respostas no Gmail |
| `/followup` | Follow-up após 3+ dias (e-mail e/ou WhatsApp) |
| `/contrato` | Contrato HTML + DOCX |

## Instalação

```
/plugin marketplace add lucasgpaixao/PROSPECTOR-DE-SITES
/plugin install prospector-de-sites@prospector-plugins
```

## Requisitos

- Claude in Chrome (Maps, deploy e WhatsApp Web), Gmail, Google Drive
- Pasta conectada no Cowork
- Hostinger com hPanel/FTP
- Python no Windows (dashboard + publicador)
